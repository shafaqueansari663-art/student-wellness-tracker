# EduPulse MVP Features (Web App)

## Student Side
- Student Login / Signup page
- Student Dashboard page
- View daily study tasks
- Mark task as complete
- Focus timer page

## Parent Side
- Parent Login / Signup page
- Parent Dashboard page
- View student progress
- View completed tasks
- View total study time

## Smart Feature
- Simple study insight / recommendation card

## Not Included in MVP
- Live GPS tracking
- School integration
- Advanced AI tutor
- Full gamification system
- Teacher dashboard
- Extra complex features
