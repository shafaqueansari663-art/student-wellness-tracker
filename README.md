# EduPulse

EduPulse is a student-first academic companion app for Gen Alpha students. It helps students stay focused, organized, and motivated through study planning, task tracking, focus sessions, gamified rewards, and AI-powered study support. At the same time, it gives parents a healthy and transparent view of their child’s academic progress, consistency, and upcoming tasks through a simple dashboard.

## Hackathon Goal
Build a working MVP where:
- Student can view and complete study tasks
- Student can start a focus session
- Parent can view academic progress
- Parent can see task completion updates
- AI provides simple study/helpful insights
